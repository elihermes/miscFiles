פתרון מודרך מקיף - תנועת רוכב אופניים

הכנה מעמיקה ומפורטת ברמת 5 יח"ל, שלב אחר שלב

💡 דגשים פדגוגיים לפני שמתחילים:

  • חלוקה לשלבים: תנועת הרוכב משתנה לאורך הדרך. נחלק את הפתרון לפרקי זמן שונים (שלבים), כאשר עבור כל שלב נחשב את הנתונים בנפרד.
  • שימוש במושג התאוצה: בפיזיקה תקנית איננו משתמשים במילה "תאוטה". כאשר גודל המהירות קטן, נאמר פשוט שכיוון התאוצה מנוגד לכיוון המהירות (תאוצה שלילית אם המהירות חיובית).
  • יחידות מידה (MKS): תמיד נעבוד במטרים (m), שניות (s), וקילוגרמים (kg). נבצע המרות יחידות בכל שלב כפי הנדרש לפני ההצבה בנוסחאות.

תרשים התנועה

התרשים הבא מציג את 4 השלבים של תנועת הרוכב. החצים הכחולים מייצגים את המהירות (v) והאדומים את התאוצה (a).

סעיף א' - חישוב הזמן הכולל למעבר 20 ק"מ

ננתח את התנועה בשלבים, משלב לשלב, ונעקוב אחר הזמן והמרחק המצטברים.

שלב 1: "בהתחלה מאיץ בתאוצה של 0.1 מטר לשנייה כל שנייה במשך דקה וחצי."

מה נתון:

  • הגוף מתחיל ממנוחה: v0 = 0
  • תאוצה: a1 = 0.1 m/s2
  • זמן: t1 = 1.5 min. נמיר לשניות: 1.5 × 60 = 90 s
  • מיקום התחלתי: נגדיר כ- x0 = 0

מה מחפשים:

את המהירות בסוף השלב (v1) ואת המרחק שעבר (x1), כדי שישמשו אותנו כנתוני התחלה לשלב הבא.

נוסחאות מדף הנוסחאות:

v = v0 + at x = x0 + v0t + ½at2

הצבה ופתרון:

חישוב המהירות בסוף השלב הראשון:

v1 = 0 + 0.1 × 90 = 9 m/s

חישוב המיקום בסוף השלב הראשון:

x1 = 0 + 0 × 90 + 0.5 × 0.1 × (90)2 = 0.05 × 8100 = 405 m

סיכום ביניים לשלב 1: לאחר 90 שניות, הרוכב נמצא במרחק של 405 מטרים ונע במהירות של 9 מטרים לשנייה.

שלב 2: "לאחר מכן נוסע במהירות אליה הגיע במשך 5 דקות."

מה נתון:

  • מהירות התחלתית לשלב זה: v1 = 9 m/s
  • מכיוון שהמהירות קבועה התאוצה היא אפס: a2 = 0 m/s2
  • פרק הזמן: Δt2 = 5 min. נמיר לשניות: 5 × 60 = 300 s
  • מיקום התחלתי לשלב זה: x1 = 405 m

מה מחפשים:

את המיקום החדש אליו הגיע הרוכב בסוף שלב זה (x2).

נוסחאות מדף הנוסחאות:

x = x0 + v0t + ½at2

הצבה ופתרון:

נציב את הזמן של השלב הנוכחי, ואת תנאי ההתחלה של השלב הנוכחי:

x2 = 405 + 9 × 300 + ½ × 0 × (300)2

x2 = 405 + 2700 = 3105 m

הזמן הכולל שעבר מתחילת התנועה הוא: t = 90 + 300 = 390 s.

שלב 3: "לאחר מכן... תאוצה של 0.05 מטר לשנייה בריבוע... במשך 30 שניות"

מה נתון:

  • מהירות התחלתית לשלב זה: v2 = 9 m/s
  • שים לב לתאוצה: נאמר שגודל המהירות קטן, לכן התאוצה מנוגדת לכיוון המהירות: a3 = -0.05 m/s2
  • פרק הזמן: Δt3 = 30 s
  • מיקום התחלתי לשלב זה: x2 = 3105 m

מה מחפשים:

את המהירות בסוף השלב (v3) ואת המיקום בסוף השלב (x3).

נוסחאות מדף הנוסחאות:

v = v0 + at x = x0 + v0t + ½at2

הצבה ופתרון:

חישוב המהירות החדשה:

v3 = 9 + (-0.05) × 30 = 9 - 1.5 = 7.5 m/s

חישוב המיקום החדש:

x3 = 3105 + 9 × 30 + 0.5 × (-0.05) × (30)2

x3 = 3105 + 270 - 0.025 × 900

x3 = 3375 - 22.5 = 3352.5 m

הזמן הכולל שעבר מתחילת התנועה הוא כעת: t = 390 + 30 = 420 s.

דגש הבנה: למרות שהמהירות יורדת, הרוכב עדיין ממשיך להתקדם קדימה (המהירות חיובית: 7.5m/s), ולכן המיקום שלו ממשיך לגדול. התאוצה השלילית רק משנה את *קצב* ההתקדמות.

שלב 4: "ממשיך לרכב במהירות קבועה... עד שעובר 20 ק"מ"

מה נתון:

  • מהירות קבועה השווה למהירות אליה הגיע: v3 = 7.5 m/s
  • תאוצה: a4 = 0 m/s2
  • מיקום התחלתי לשלב זה: x3 = 3352.5 m
  • מיקום סופי הנדרש: 20 ק"מ. נמיר למטרים: xfinal = 20 × 1000 = 20,000 m

מה מחפשים:

את פרק הזמן האחרון הנדרש (Δt4) ומתוכו את הזמן הכולל מתחילת התנועה.

נוסחאות מדף הנוסחאות:

x = x0 + v0t + ½at2

הצבה ופתרון:

נחשב את המרחק שנותר לרוכב לעבור:

Δx4 = xfinal - x3 = 20000 - 3352.5 = 16,647.5 m

נציב במשוואת המיקום (כאשר a=0 ומרחק התחלתי הוא 0 עבור השלב הספציפי הזה):

16647.5 = 0 + 7.5 × Δt4 + 0

Δt4 = 16647.5 / 7.5 = 2219.67 s

הזמן הכולל מתחילת התנועה:
ttotal = 420 + 2219.67 = 2639.67 s

סעיף ב' - כתיבת משוואת התנועה

מהי משוואת תנועה כאשר התנועה מורכבת משלבים?

משוואת התנועה היא למעשה הפונקציה המתארת את המיקום (x) כפונקציה של הזמן הכולל שעבר (t). מכיוון שהתאוצה משתנה בין השלבים, לא נוכל לכתוב נוסחה אחת בלבד, אלא נכתוב פונקציה מפוצלת (Piecewise function) המגדירה משוואה שונה לכל פרק זמן.

נשתמש בנוסחה המוכללת לדף הנוסחאות, המאפשרת התחלת זמן לא מאפס: x(t) = x0 + v0(t - t0) + ½a(t - t0)2

סיכום הנתונים למשוואה:

שלב תחום הזמן (שניות) x0 (מטרים) v0 (m/s) a (m/s2)
1 0 ≤ t < 90 0 0 0.1
2 90 ≤ t < 390 405 9 0
3 390 ≤ t < 420 3105 9 -0.05
4 420 ≤ t ≤ 2639.67 3352.5 7.5 0
x(t) =
0.05t2 (0 ≤ t < 90)
405 + 9(t - 90) (90 ≤ t < 390)
3105 + 9(t - 390) - 0.025(t - 390)2 (390 ≤ t < 420)
3352.5 + 7.5(t - 420) (t ≥ 420)

המחשה גרפית: מהירות כפונקציה של הזמן

למרות שלא נתבקשנו לצייר גרף, שרטוט גרף מהירות-זמן עוזר מאוד להבין את הבעיה. השיפוע בגרף הוא התאוצה (בחלקים האופקיים התאוצה היא אפס), והשטח הכלוא מתחת לגרף הוא ההעתק (במקרה שלנו, 20,000 מטר).
